# 기본 LMS 코어

## 목적
AI 기능이 없어도 성립하는 MyWayClass의 기본 LMS 범위를 정의한다.

## 범위
- 로그인과 권한
- 강의 목록과 상세
- 수강 신청과 진도
- 강의 자료와 공지
- 관리자 기본 기능

## 입력
- 사용자 계정 정보
- 강의 메타데이터
- 수강 상태
- 운영자 관리 정보

## 출력
- 학습 가능한 강의 화면
- 수강 상태와 진도 정보
- 공지와 자료 접근 결과

## 규칙
- 기본 LMS는 AI 레이어 없이도 동작해야 한다.
- 코어 기능은 강의 관리와 학습 진행에 집중한다.
- 화면과 API는 강의, 수강, 관리의 경계를 넘지 않게 설계한다.
- 숏폼, STT, RAG, 챗봇은 이 문서의 범위를 넘는 AI 기능으로 분리한다.

## 핵심 기능
- 회원 가입과 로그인
- 역할 기반 접근 제어
- 강의 목록, 검색, 상세
- 수강 신청, 수강 해제, 진도 저장
- 강의 자료 조회
- 공지 확인
- 관리자용 강의 등록과 편집

## 예외 상황
- AI 기능이 비활성화되어도 강의 조회와 수강은 가능해야 한다.
- 데모용 강의는 자료와 진도를 단순화할 수 있다.

## 실패 모드
- AI 기능이 꺼지면 LMS 자체가 동작하지 않는 경우
- 강의와 수강의 책임이 섞이는 경우
- 관리자 기능이 일반 학습자 화면에 노출되는 경우

## 검증 기준
- AI 없이도 강의 등록, 조회, 수강, 진도 흐름이 가능하다.
- 학습자는 기본 LMS만으로 수업을 따라갈 수 있다.
