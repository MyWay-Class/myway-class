# 프로젝트 문서 색인

## 목적
MyWayClass의 기능, 기획, 구현 기준 문서를 한곳에서 찾기 쉽게 정리한다.

## 문서 허브
- `docs/README.md`는 전체 문서의 출발점이다.
- `docs/project/`는 프로젝트 핵심 문서를 담는다.
- `docs/context/`는 시스템 구조와 환경 설정을 담는다.
- `docs/conventions/`는 코드와 문서의 공통 규칙을 담는다.
- `docs/structure/`는 기능별 상세 구조를 담는다.
- `docs/templates/`는 새 문서를 만들 때 복사할 양식을 담는다.
- `docs/dev-logs/`는 작업 중 메모와 진행 기록을 담는다.
- `docs/ai-context/`는 AI 협업 규칙을 담는다.
- `docs/ops/`는 짧은 운영용 보조 문서를 담는다.
- `frontend/docs/`는 프론트엔드 전용 문서를 담는다.
- `backend/docs/`는 백엔드 전용 문서를 담는다.
