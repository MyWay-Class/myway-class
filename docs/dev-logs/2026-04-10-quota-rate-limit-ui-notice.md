# quota 및 rate limit 안내 UI 보강

## 배경
- 백엔드에는 AI 로그인, quota, rate limit이 이미 있었지만, 화면에서 차단 사유와 남은 사용량을 더 친절하게 보여줄 필요가 있었다.

## 변경 내용
- AI 채팅 화면에 공개 테스트 정책 안내와 요청 상태 배너를 추가했다.
- AI 요약 화면에 공개 테스트 범위 안내 배너를 추가했다.
- 미디어 파이프라인 화면에 공개 테스트 정책, quota, rate limit, STT 3분 제한 안내를 추가했다.
- API 응답의 실패 본문과 헤더를 프론트에서 읽어 배너에 활용하도록 공통 request 처리를 확장했다.

## 검증
- `npm run verify`를 통과했다.
