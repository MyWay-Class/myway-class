# 2026-04-13 media pipeline role-based status board A/B comparison

## 변경 요약
- `#151` 미디어 파이프라인 상태판의 권한별 노출을 A/B 워킹트리로 비교했다.
- A안은 기존 상태판을 유지하면서 역할에 따라 내부 상세 섹션만 숨기는 최소 변경이었다.
- B안은 상단에 학습자용 요약 패널을 두고, 하단에 관리자/강사용 상세 패널을 분리하는 방식이었다.

## 선택 결과
- B안을 채택했다.

## 선택 이유
- 요약 정보와 운영 정보를 한 카드 안에서 토글하는 방식보다, 화면 상에서 먼저 봐야 할 정보와 숨겨야 할 정보를 분리하는 편이 덜 복잡하다.
- 학습자에게는 업로드/전사/요약 상태만 보여주고, 관리자와 강사는 상세 상태판을 아래에서 확인하는 흐름이 더 명확하다.
- 역할별 안내 문구와 액션 위치가 분리되어 있어서 이후 학생용/관리자용 화면 재사용도 쉽다.

## 제외한 방향
- A안은 구현이 간단하지만, 같은 상태판 안에서 섹션만 숨겨서 구조가 덜 분명하다.
- 정보가 많은 페이지일수록 토글식 숨김보다 섹션 분리가 사용성에 유리하다.
