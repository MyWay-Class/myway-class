# 2026-04-10 강의 제작 스튜디오 UI/UX 고도화

## 변경 이유
- 강사용 강의 제작 화면이 부족해서 수강 인원, 강의실, 강의 형태, 과제, 시험, 퀴즈를 한 번에 다루는 UI가 필요했음.
- 기존 강의/숏폼 화면은 있었지만 제작 초안 중심의 진입점이 약했음.

## 변경 내용
- `lecture-studio` 페이지를 추가해 강사용 제작 진입점을 만들었다.
- 강의 선택, 기본 정보, 운영 방식, 과제/시험/퀴즈, 목차, 연결 자료, AI 보조 옵션을 분리해서 배치했다.
- 우측 미리보기에서 수강 인원, 강의실, 평가 정책, AI 연동 상태를 바로 보이도록 구성했다.
- 저장/발행 API는 다음 이슈에서 연결되도록 UI만 먼저 정리했다.

## 검증
- `npm run verify` 통과
