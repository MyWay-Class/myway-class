# 2026-04-10 숏폼 미리보기 모달 닫힘 버그 수정

## 변경 이유
- 커뮤니티 화면에서 숏폼 미리보기를 닫아도 클릭이 막힌 것처럼 보이는 문제가 있었음.
- 닫기 이후에도 첫 아이템으로 다시 되돌아가는 구조라 모달이 사실상 계속 유지됐음.

## 변경 내용
- `CommunityPage`에서 모달용 선택 상태와 상세 프리뷰 상태를 분리했다.
- 모달은 선택한 아이템이 있을 때만 렌더되도록 바꿨다.
- `ShortformPreviewModal`에 배경 클릭 닫기와 `Escape` 닫기를 추가했다.
