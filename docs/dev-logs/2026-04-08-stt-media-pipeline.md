# STT와 미디어 파이프라인

## 변경 내용
- `backend/src/routes/media.ts`를 추가해 트랜스크립트, 요약 노트, 오디오 추출, 파이프라인 상태 조회를 연결했다.
- `packages/shared`에 트랜스크립트/노트/추출/파이프라인 타입과 데모 로직을 추가했다.
- API 계약 문서와 STT 파이프라인 문서를 현재 구현에 맞게 갱신했다.

## 검증
- `npm run build` 통과

## 메모
- 미디어 파이프라인은 데모 데이터 기반으로 동작하지만, 계약은 실제 운영 형태에 맞춰 분리했다.
- 요약과 타임라인은 같은 lecture 원문을 기준으로 생성되도록 정리했다.
