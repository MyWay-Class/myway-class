# 2026-04-06 기본 LMS 코어 작업 로그

## 변경 이유
- 기본 LMS 기능을 먼저 만들어야 강의 숏폼화와 커스텀 강의 기능을 얹을 수 있다.
- 코드와 문서가 함께 움직이도록 먼저 데이터 계약과 화면/API 흐름을 맞췄다.

## 변경 내용
- `packages/shared/src/index.ts`에 기본 LMS 도메인 타입과 데모 데이터, 조회 헬퍼를 추가했다.
- `backend/src/index.ts`와 `backend/src/routes/*`에 대시보드, 강의, 수강 신청, 강의 조회 API를 추가했다.
- `frontend/src/App.tsx`와 `frontend/src/lib/api.ts`에 LMS 대시보드, 강의 목록, 강의 상세, 수강 신청 화면을 추가했다.
- `frontend/index.html`, `frontend/tsconfig.json`, `backend/tsconfig.json`을 추가해 앱 진입점을 정리했다.

## 검증 상태
- 코드 구조와 계약은 수동 검토했다.
- 실제 빌드와 실행은 의존성 설치 이후 확인이 필요하다.
