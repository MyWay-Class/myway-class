# Dashboard, Chat, Duration Fix

## 개요
- 배포 버전에서 확인된 세 가지 문제를 한 번에 정리했다.
- `수강 중인 강의` 카드가 상세 화면으로 이어지지 않는 문제, 챗봇 텍스트의 깨짐/`???` 노출, 실제 영상 길이와 transcript/숏폼 길이 불일치를 수정했다.

## 반영 내용
- 학생 대시보드의 강의 카드를 누르면 `강의 상세` 화면으로 이동하도록 연결했다.
- AI 챗봇 출력과 추천 질문/근거 텍스트에서 깨진 문자와 `???` 패턴을 정리했다.
- STT callback 시 실제 `audio_duration_ms`를 transcript 생성에 넘겨, 숏폼과 타임스탬프가 실제 영상 길이를 따르도록 맞췄다.

## 의도
- 카드 클릭은 선택만 해서는 UX가 끊기므로, 바로 상세 화면으로 이어져야 한다.
- 챗봇은 강의 맥락을 읽는 기능이기 때문에, 깨진 문자열이나 placeholder는 곧바로 신뢰도 하락으로 이어진다.
- 숏폼과 스크립트는 실제 영상 길이에 맞아야 하며, 메타데이터가 아닌 실제 추출 길이를 우선해야 한다.
