# v0.4.0 레퍼런스형 강의 탐색 Hero/필터 반영 - B안

## 변경 요약
- 강의 탐색 화면을 레퍼런스 구조에 맞춰 Hero, 검색, 카테고리 필터, 상태 탭으로 재구성했다.
- 카드 목록과 우측 상세 패널을 분리해 유지하면서도 첫 화면 탐색 경험을 개선했다.

## B안 선택 이유
- 레퍼런스의 핵심은 첫 화면에서 검색 의도와 필터 진입점을 명확히 보여주는 구조였다.
- 기존 `CoursesPage`에 그대로 덧붙이는 A안보다, Hero / Filter / Card / Detail을 분리한 B안이 디자인 일치도와 유지보수성 모두 더 좋았다.
- 파일이 길어지기 전에 컴포넌트를 분리하는 편이 앞으로의 UX 수정에도 유리했다.

## 변경 내용
- Hero 배너에 검색 입력, 전체/수강중/수강가능 요약, 추천 태그를 배치했다.
- 카테고리 아이콘 필터와 상태 탭을 추가했다.
- 강의 카드에 태그 노출과 선택 강조를 정리했다.
- 우측 상세 패널은 차시 타임라인, 공지, 자료 탭을 유지하면서 레퍼런스 느낌의 상단 요약 영역으로 다듬었다.

## 확인 사항
- `npm exec --workspace @myway/frontend -- tsc -p tsconfig.json --noEmit`
