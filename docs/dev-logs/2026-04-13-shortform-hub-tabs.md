# Shortform Hub Tabs

## 개요
- `숏폼 제작`, `내 숏폼`, `숏폼 커뮤니티`를 각각 따로 들어가는 구조에서, 하나의 진입점 안에 탭으로 묶었다.
- 학생의 숏폼 제작은 `수강 중인 강의`만 선택할 수 있도록 제한했다.

## 반영 내용
- `ShortformHubPage`를 추가해 제작 / 내 숏폼 / 커뮤니티를 하나의 화면에서 전환하도록 바꿨다.
- 사이드바와 대시보드 진입점은 `숏폼` 하나로 정리했다.
- `shortform`, `my-shortforms`, `community`는 모두 같은 허브로 연결되도록 라우터를 정리했다.
- 학생의 제작 탭은 수강 중인 강의가 없으면 안내만 보여주고, 수강 중인 강의가 있을 때만 제작 흐름을 시작한다.

## 의도
- 숏폼 기능이 여러 페이지로 갈라져 있으면 사용자가 흐름을 잃기 쉽다.
- 제작, 보관함, 커뮤니티는 같은 맥락에서 이어지는 기능이므로 하나의 허브로 묶는 편이 더 자연스럽다.
- 학생의 제작 범위는 수강 중인 강의로 제한해야 데이터와 UX가 일치한다.
