# 2026-04-12 course create A/B compare

## Issue
- #152 [교수/강사용 강의 개설 흐름 추가](https://github.com/MyWay-Class/myway-class/issues/152)

## A안
- `CoursesPage` 안에 새 강의 생성 카드를 바로 붙였다.
- 장점: 진입이 빠르고 기존 강의 목록 흐름에서 바로 개설할 수 있다.
- 단점: 개설 기능이 탐색 화면 안에 섞여서, 배포 버전에서의 "새 강의 개설" 진입점이 덜 분명하다.

## B안
- `course-create` 전용 페이지를 만들고, 목록 화면에서는 개설 페이지로 이동하는 CTA만 제공한다.
- 생성 성공 후 `lecture-studio`로 바로 넘겨서 다음 작업 흐름이 끊기지 않게 했다.
- 기본 자료/공지 초안과 첫 차시 생성까지 함께 묶어서, 실제 운영 흐름에 더 가깝게 만들었다.

## 선택 이유
- 배포 버전에서 교수/강사/운영자가 "새 강의 개설"을 명확하게 인지하고 사용할 수 있어야 한다.
- 내부 상태판보다 사용자가 직접 쓰는 생성 진입점이 분리되는 편이 UX가 더 깨끗하다.
- 생성 후 강의 제작 스튜디오로 이어지는 흐름이 B안에서 더 자연스럽다.

## 검증
- `npm run verify` 통과
- 로컬 새 강의 생성 후 목록/상세/스튜디오 연결 경로 확인
