# 2026-04-10 미디어/STT 공개 테스트 정책 정리

## 변경 이유
- 공개 테스트에서 긴 영상 실시간 처리를 열어두면 무료 티어와 운영 리스크가 커진다.
- 운영자 사전 처리 흐름과 일반 사용자 체험 범위를 문서로 분리해야 했다.

## 변경 내용
- `docs/project/14-ai-layer.md`에 공개 테스트용 AI/STT 범위와 provider 정책을 정리했다.
- `docs/project/07-stt-media-pipeline.md`에 `media/transcribe` 3분 제한과 관리자 전용 업로드 정책을 적었다.
- 긴 영상은 운영자가 미리 처리하고, 일반 사용자는 준비된 transcript와 짧은 STT 체험만 쓰는 기준을 명시했다.

## 검증
- 문서 변경만 반영했다.
