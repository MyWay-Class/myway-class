# 숏폼 커뮤니티 미리보기 강화 - B안

## 변경 요약
- 숏폼 커뮤니티 화면을 카드 미리보기와 상세 모달 구조로 재구성했다.
- 커뮤니티 데이터를 실제 숏폼 항목으로 불러와 카드 안에서 클립 목록, 길이, 조회 수를 먼저 보여준다.

## B안 선택 이유
- 레퍼런스는 카드 목록만 보여주는 수준이 아니라, 카드 안에서 클립 구성을 먼저 보여주고 상세 모달에서 전체 맥락을 확인하는 흐름이었다.
- A안은 단순한 미리보기 확장형이었지만, 타입 체크에서 문법 문제가 생겼고 구조적으로도 레퍼런스와의 일치도가 낮았다.
- B안은 Hero / Card / Modal을 분리해 파일이 길어지지 않고 이후 재사용하기 좋았다.

## 변경 내용
- 커뮤니티 Hero에 항목 수, 클립 수, 조회 수 요약을 추가했다.
- 카드 안에 상위 3개 클립을 표시했다.
- 카드 클릭 시 상세 패널과 미리보기 모달이 함께 보이도록 구성했다.
- 기존 추천/코스 데이터는 커뮤니티 데이터가 비어 있을 때만 fallback으로 사용한다.

## 확인 사항
- `npm exec --workspace @myway/frontend -- tsc -p tsconfig.json --noEmit`
