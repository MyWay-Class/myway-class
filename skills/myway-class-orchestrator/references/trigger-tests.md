# Trigger Tests for myway-class-orchestrator

## should-trigger
1. myway-class 프론트 버그 수정하고 백엔드도 맞춰줘
2. 수강신청 API 필드 바뀐 거 화면까지 동기화해줘
3. 이전 결과 기반으로 로그인 흐름만 다시 실행해줘
4. 결제 실패 케이스까지 포함해서 전체 보완해줘
5. 프론트/백엔드 둘 다 리팩터링하고 검증해줘
6. 어제 작업 이어서 부분 수정만 해줘
7. 응답 코드 정책 변경 반영해서 UI 에러 처리 업데이트해줘
8. 업데이트된 요구사항 기준으로 다시 실행해줘
9. 내맘대로Class 기능 추가하고 QA까지 돌려줘
10. API 연동 깨진 구간만 잡아줘

## should-not-trigger
1. 현재 서울 시간 알려줘
2. git 기본 명령어 설명해줘
3. 이 문장을 영어로 번역해줘
4. 파이썬 리스트 컴프리헨션 예시 보여줘
5. 오늘 날씨 요약해줘
6. 일반적인 면접 질문 5개만 줘
7. 2차 방정식 공식 알려줘
8. 마크다운 표 문법만 설명해줘
9. 랜덤 숫자 1개 뽑아줘
10. JSON이 뭐야?
